Política de Privacidade

Sua privacidade é nossa prioridade. Este documento descreve quais dados coletamos, como utilizamos e como protegemos suas informações.

📅 Última atualização: Março de 2026 📦 Versão: v1.5.0
01 — Introdução

Quem Somos

O Architect é um bot para Discord desenvolvido e mantido por Alzhayds. Esta Política de Privacidade explica como tratamos os dados necessários para o funcionamento do bot.

Ao utilizar o Architect, você concorda com as práticas descritas neste documento.

◆ Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins comerciais.
02 — Dados Coletados

O Que Coletamos

O Architect coleta apenas os dados estritamente necessários para suas funcionalidades:

DadoFinalidadeObrigatório
ID do ServidorIdentificar o servidor para salvar e recuperar backups✅ Sim
Nomes de canais e cargosSalvar a estrutura do servidor no sistema de backup✅ Sim
Permissões de cargos e canaisRestaurar as configurações exatas após um nuke✅ Sim
Últimas 50 mensagens por canalRestaurar histórico recente quando /backup é executado⚙️ Opcional
ID de usuários no audit logDetectar atividade suspeita no sistema anti-nuke⚙️ Somente com proteção ativa
03 — Não Coletamos

O Que Não Coletamos

Para sua total tranquilidade, confirmamos que o Architect nunca coleta:

  • Senhas, tokens ou credenciais de qualquer tipo
  • Informações pessoais dos membros como nome real, e-mail ou telefone
  • Dados de pagamento ou informações financeiras
  • Conteúdo de mensagens diretas (DMs) entre usuários
  • Histórico completo além das últimas 50 mensagens por canal
  • Localização, endereço IP ou dados de dispositivo
04 — Uso dos Dados

Como Utilizamos os Dados

Os dados coletados são utilizados exclusivamente para:

  • Salvar e restaurar a estrutura completa do seu servidor Discord
  • Detectar e alertar sobre atividades suspeitas de nuke quando a proteção está ativa
  • Melhorar a qualidade e precisão das funcionalidades do bot
◆ Os dados são armazenados localmente nos servidores do Architect e nunca são compartilhados com serviços de terceiros para fins analíticos ou publicitários.
05 — Retenção

Por Quanto Tempo Guardamos

Os dados são armazenados enquanto o Architect estiver ativo no seu servidor:

  • Backups — mantidos até que você execute um novo /backup ou remova o bot
  • Dados do anti-nuke — armazenados apenas em memória durante a sessão ativa, não persistem após reinicialização

Ao remover o Architect do seu servidor, os dados associados podem ser excluídos mediante solicitação.

06 — Seus Direitos

Seus Direitos

Você tem o direito de:

  • Acesso — solicitar quais dados temos do seu servidor
  • Exclusão — solicitar a remoção completa dos dados do seu servidor
  • Portabilidade — solicitar seus dados em formato legível
  • Oposição — remover o bot do servidor a qualquer momento, encerrando a coleta

Para exercer qualquer um desses direitos, entre em contato pelo servidor oficial de suporte do Architect.

07 — Segurança

Segurança dos Dados

Adotamos medidas razoáveis para proteger os dados armazenados pelo Architect:

  • Armazenamento local sem exposição pública dos dados
  • Sem integração com serviços de analytics ou rastreamento
  • Acesso restrito apenas às funcionalidades essenciais do bot

Em caso de incidente de segurança relevante, notificaremos os usuários afetados pelo servidor de suporte.

08 — Contato

Dúvidas e Solicitações
